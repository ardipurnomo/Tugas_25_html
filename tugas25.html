Mengenal Semantic Element Di HTML5

  • article

    • Ketika kita ingin membuat layout untuk menampilkan block artikel-artikel yang ada di website kita,tag artikel bisa kita gunakan dengan element header dan footer.

    Karen tag artikel memiliki type block, artinya elemen ini memiliki lebar sesuai lebar browser jika kita tidak menentukan lebarnya.

  • Aside

    • Element ini bisa kita gunakan untuk melengkapi layout utama. Misal kita memiliki halaman blog maka kita bisa menambahkan aside di bawah bagian postingan untuk menjelaskan identitas penulis artikel.

    Karena aside memiliki sifat atau type block maka lebarnya sesuai page jika kita menentukan lebarnya
